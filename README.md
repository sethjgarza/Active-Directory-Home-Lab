# Windows Server 2022 Active Directory Home Lab

## Overview

Built a Windows Server 2022 Active Directory environment in VirtualBox to gain hands-on experience with domain administration, Group Policy, file sharing, permissions, and user management.

## Technologies Used

- Windows Server 2022
- Windows 11 Pro
- Active Directory
- DNS
- Group Policy
- VirtualBox

## Skills Demonstrated

- Installed and configured Active Directory Domain Services
- Promoted a Domain Controller
- Created users and security groups
- Configured password and account lockout policies
- Joined Windows 11 clients to the domain
- Created SMB shares
- Configured NTFS and share permissions
- Configured home folders and mapped drives
- Troubleshot authentication and file-sharing issues

## Lab Environment

**Domain:** leapfrog.com

**Domain Controller:** TX-DC-01

**Client:** Windows 11 Pro
