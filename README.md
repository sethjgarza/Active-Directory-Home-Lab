# Windows Server 2022 Active Directory Home Lab

## Overview

Built a Windows Server 2022 Active Directory environment using VirtualBox.

## Technologies Used

- Windows Server 2022
- Windows 11 Pro
- Active Directory Domain Services
- DNS
- Group Policy
- SMB File Sharing
- NTFS Permissions
- VirtualBox

## Lab Environment

Domain: leapfrog.com

Domain Controller: TX-DC-01

Client Machine: Windows 11 Pro

## Skills Demonstrated

- Installed Active Directory Domain Services
- Promoted a Domain Controller
- Created users and groups
- Configured Group Policy
- Configured password and account lockout policies
- Created SMB shares
- Configured NTFS permissions
- Configured home folders and mapped drives
- Joined Windows 11 client to the domain
- Troubleshot file sharing and permissions issues
